# createGame - Pixelize

```js
import { createGame } from "pixelize-engine";

const game = createGame(/* config */);
```

`createGame(`_`config`_`)` will create and return a new [`Game`](/docs/game).

Paramaters:

- `config`: The [`Config`](/docs/config) for the game.
